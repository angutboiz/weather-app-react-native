# Weather Appication Using React Native

Weather Appication using framework React Native for smartphones

## Installation

```bash
git clone 
npm run web
```
## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
